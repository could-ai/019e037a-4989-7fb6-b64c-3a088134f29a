# Team-Präsentation App

Eine interaktive, lustige Flutter-App im Stil einer Präsentation (PowerPoint-Ersatz) zur Vorstellung von Teammitgliedern. 

## Features
- **Startbildschirm:** Einladender Intro-Screen mit fröhlichen Farben.
- **Interaktive Slides:** Eine `PageView`, durch die man per Wischen oder über Pfeiltasten navigieren kann.
- **Mitglieder-Profilkarten:** Bunte Karten mit Name, Rolle, Emoji und Fun Fact.
- **End-Screen:** Ein Abschluss-Slide mit der Möglichkeit, die Präsentation neu zu starten.

## Tech Stack
- Flutter (Dart)
- Plattformübergreifend (iOS, Android, Web, Desktop)

## Setup & Run
Um die App lokal auszuführen:
```bash
flutter pub get
flutter run
```
